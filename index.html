Sobre Mim

Meu nome é Leonardo Moussallem

Atualmente, estou me graduando em Engenharia da Computação com previsão de conclusão em 2026. Além disso, atuo como desenvolvedor de software, focando em tecnologias como JavaScript, Node.js, React.js e React Native. Tenho experiência em desenvolvimento de produtos de dados utilizando técnicas de Machine Learning para resolver desafios empresariais no setor privado. Recentemente, meu trabalho mais significativo foi o desenvolvimento de um aplicativo de consultas empresariais para uma empresa de assesoria, utilizando JavaScript e React Native. Esta solução foi projetada para melhorar significativamente a eficiência operacional. Como profissional, tenho um forte compromisso com a entrega rápida e a qualidade das soluções de software. Priorizo uma abordagem prática na resolução de problemas de negócios, combinando sólido embasamento teórico com habilidades avançadas em desenvolvimento de software. Neste ponto da minha carreira, estou confiante em contribuir como Engenheiro de Software em diversas empresas, aplicando meu conhecimento em JavaScript, Node.js, React.js, React Native, além de ferramentas de desenvolvimento como Git, Scrum e Lean. Minha formação em Engenharia da Computação me capacita a pensar de forma inovadora e encontrar soluções eficazes e criativas para desafios empresariais. Minhas habilidades analíticas incluem SQL para manipulação de dados, Python para desenvolvimento de algoritmos de Machine Learning, e ferramentas como Tableau e Looker para visualização de dados. Além disso, tenho experiência com bancos de dados como MySQL, PostgreSQL e MongoDB, e em implementação de pipelines de dados usando Airflow, Logstash e Nifi. Estou focado em continuar aprendendo e crescendo profissionalmente, sempre buscando oportunidades para aplicar minhas habilidades técnicas e de análise em projetos desafiadores que gerem valor real para as empresas.
